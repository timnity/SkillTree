# Express Server

[express](http://expressjs.com/zh-cn/) 是Node.js上最受欢迎的Web架构之一，它可以快速方便的实现Web应用服务、API服务……是node.js后端入门的基础知识之一，前端也应了解并熟练掌握

## 安装
`npm install express --save`
